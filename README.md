# Quant-Chem-Bio-oil-ECH
This repository contains public data corresponding to the manuscript
titled "Quantum Chemical Insight into Electrochemical Upgrading of
Bio-oil"


In the "Preferred Configurations" folder, *.xyz files are provided for
the optimized geometries of each species in the preferred configurations on
Pt(100), Pt(110), Pt(111), and Ru(0001).
